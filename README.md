# Hassan Raza Portfolio

A responsive personal portfolio website for Hassan Raza, showcasing AI engineering, data science, desktop ERP, and web development projects.

## Overview

This portfolio is built as a static website using HTML and CSS. It highlights featured client work, data science dashboards, contact information, social links, and a downloadable CV.

## Features

- Responsive landing page with hero section
- Featured project cards with status badges and project metadata
- Data science project links
- Downloadable CV
- Contact section with email, phone, LinkedIn, GitHub, and Kaggle links
- Modern dark UI with gradient accents

## Tech Stack

- HTML5
- CSS3
- Font Awesome icons
- Google Fonts

## How to Run

Open `index.html` directly in a browser.

For a local development server, you can also run:

```bash
python -m http.server 8000
```

Then visit:

```text
http://localhost:8000
```

## Customization

- Update portfolio content in `index.html`
- Update colors, spacing, and responsive layout in `style.css`
- Replace profile image by updating `me.png`
- Replace project screenshots inside the `assets/` folder
- Replace `Hassan_Cv.pdf` to update the downloadable CV
